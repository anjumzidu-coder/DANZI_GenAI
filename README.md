# DANZI GenAI – NSI Leadership Update Agent (CrewAI)

A portfolio-ready **CrewAI multi-agent system** that converts raw NSI project notes into
**leadership-ready executive updates**.

## What it generates
From one input file (raw NSI notes), it produces:
- Executive summary (VP/Director ready)
- Status snapshot
- Risks & dependencies with mitigation (Risk–Impact–Mitigation–Owner)
- Clear **Leadership Asks**
- 60–90 second talk track

## Why this project
In delivery readiness / program management work, the hardest part is turning messy updates into
decision-oriented leadership communication. This repo automates that.
